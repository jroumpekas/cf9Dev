# 📚 cf9Dev – Java Programming Exercises

Το repository **cf9Dev** περιλαμβάνει την προσωπική σου υλοποίηση των ασκήσεων Java για το πρόγραμμα **Coding Factory 9**.  
Εδώ οργανώνονται όλα τα chapters (ch1, ch2, ch3, ...) και οι ασκήσεις πρακτικής.

---

## 📂 Δομή του Project

Η δομή των φακέλων είναι ως εξής:

src/
└── gr/
└── aueb/
└── cf/
└── cf9/
├── ch1/ # Βασικές εντολές, Hello World, απλές πράξεις
├── ch2/ # Έλεγχος ροής, συνθήκες, λογικές εκφράσεις
├── ch3/ # Επαναλήψεις (for, while, do-while)
├── ch4/ # Πίνακες (arrays)
├── ch5/ # Μέθοδοι / Functions
├── ch6/ # Object-Oriented Programming
├── ch7/ # Exceptions
├── ch8/ # Collections
├── ch9/ # File I/O
└── review/challenges/ # Extra challenges για εξάσκηση


Κάθε φάκελος περιέχει αρχεία `.java` με ολοκληρωμένες λύσεις.

---

## 🧪 Πώς να τρέξεις τα προγράμματα

1. Άνοιξε το project στο **IntelliJ IDEA**  
2. Πήγαινε στο αρχείο `.java` που θέλεις να εκτελέσεις  
3. Πάνω από τη `main` μέθοδο πάτα το ▶️ (Run)

## 📝 .gitignore (σημαντικό)

Για να κρατήσεις το repository καθαρό, χρησιμοποιείται `.gitignore` που αποκλείει:

- `.idea/` (IntelliJ settings)
- `out/` (compiled files)
- `.iml` αρχεία
- temporary/system files
